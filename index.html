Emotion is a performant and flexible CSS-in-JS library. Building on many other CSS-in-JS libraries, it allows you to style apps quickly with string or object styles. It has predictable composition to avoid specificity issues with CSS.

Advantages :)

  • Encapsulate styles within component...
  • ...but allow to reuse some of them.
  • you can pass props and styles can be easily reused
  • works with js features like loops, conditional statements, variables
  • We can easily found not used styles
  • Easy to move between components
  • Super performant - style per component
  • Supports tree shaking

Disadvantages :(

  • Provides one more layer of abstraction
  • Learning Curve - can be hard for begginers
  • In big projects can produce big output
  • How to name our styled components?(*)
  • Browser debugging(*)
